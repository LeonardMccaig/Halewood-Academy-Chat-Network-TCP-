How to use:

TO OPEN A SERVER: 

First register with ngrok Using the link here https://ngrok.com/

once you have registered Open the directory in cmd and run the command ngrok.exe tcp [insert port here e.g 8008]

copy the 'forwarding' link provided by ngrok e.g 2.tcp.eu.ngrok.io:11360 - this will be your link for client to connect to the server with 

now simply replace enter the ngrok link when prompted and you will be connected to the chat server


TO JOIN THE CHAT:

simply run the program and Join the chat room and enjoy!
